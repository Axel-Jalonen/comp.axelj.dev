ReadComp Documentation

What is ReadComp?

ReadComp is a compression algorithm that aims to be a human readable way to reduce file size. There are a variety of applications for such an algorithm that systematically reduces file size. These include:

  • Reducing the size of a file before writing it on a study sheet
  • Reducing the size of a file before storing it on a disk
  • Reducing the size of a file before writing it by hand
