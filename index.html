ReadComp offers variable compression that aims to be human readable. Unlike other human readable compression algorithms that aim to reduce reading strain without a focus on file size reduction, ReadComp aims to provide the maximum amount of file size compression, fewer bytes is desirable, while still remaining readable. You can modulate the parameters to achieve more readability or more compression. In the output box you can view the character reduction as well as the new word & character count.
